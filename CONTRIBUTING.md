# Contributing to n8n-manager-mcp

Thanks for your interest in contributing!

## Getting Started

1. Fork and clone: `git clone https://github.com/YOUR_USERNAME/n8n-manager-mcp.git`
2. Install: `npm install`
3. Build: `npm run build`
4. Requires: Node.js 18+, a running n8n instance

## How to Contribute

- **Bug fixes**: Open an Issue or submit a PR directly
- **New MCP tools**: Extend n8n API coverage
- **Documentation**: Always appreciated

### Good First Issues

Look for [`good first issue`](https://github.com/ellmos-ai/n8n-manager-mcp/labels/good%20first%20issue) labels.

## Guidelines

- TypeScript with strict types
- One feature/fix per PR
- Test against a local n8n instance before submitting
